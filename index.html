My Projects / Work

Seasonal Analytics Reporting System

View on GitHub →

High-Impact Business Intelligence Upgrade for McKaylan Motor Engineering

The Problem

Previously, management relied on static, single-dimensional reports to gauge seasonal performance. This resulted in missed opportunities for optimizing labor allocation and marketing spend.

The Solution

A dynamic, multi-vector analytics system that transforms raw operational data into actionable strategic insights in real-time. It allows leadership to simultaneously track operational demand, workforce efficiency, and client acquisition trends through a secure, unified dashboard.

Technical Architecture

  • Core ASP.NET Web Forms (.NET Framework 4.8)
  • Language C# (with modern pattern matching)
  • Database MS SQL Server (via secure ADO.NET)
  • Security Parameterized `SqlCommand` (prevents SQLi)
  • Frontend HTML5/CSS3, JavaScript & Chart.js
